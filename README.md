# 贪吃蛇小游戏

标签（空格分隔）： bootstrap

---

**原型方法封装的贪吃蛇小游戏，使用了bootstrap框架来布局和模态框插件，利用localStorage存储排行榜数据**

## 游戏说明

 - 一进入页面为游戏说明，介绍游戏规则
 - 点击`Play Now`进入游戏画面

## 参数配置

默认参数保存在`snake.js`的defaults，以下参数可以更改：

    cellWidth: 20,        //单元格的大小
    speed: 2,            //蛇的移动速度
    initialLength: 3,   //蛇的初始长度
